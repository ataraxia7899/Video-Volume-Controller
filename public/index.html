Video Volume Controller 확장 프로그램

저희 "Video Volume Controller" 브라우저 확장 프로그램을 이용해 주셔서 감사합니다. 본 확장 프로그램은 사용자의 비디오 볼륨을 편리하게 제어하기 위해 설계되었습니다.

데이터 수집 및 사용

저희 "Video Volume Controller" 확장 프로그램은 어떠한 개인 식별 정보도 수집, 저장 또는 전송하지 않습니다. 확장 프로그램은 전적으로 사용자의 로컬 환경에서 작동하며, 외부 서버와 통신하지 않습니다.

특히, 다음 범주의 사용자 데이터는 수집되지 않습니다:

  • 개인 식별정보 (이름, 이메일 주소, IP 주소 등)
  • 건강 정보
  • 금융 및 결제 정보
  • 인증 정보
  • 개인적인 커뮤니케이션
  • 위치
  • 웹 기록
  • 사용자 활동
  • 웹사이트 콘텐츠

확장 프로그램의 모든 기능은 사용자의 브라우저 내에서만 처리되며, 사용자의 데이터는 사용자의 기기를 벗어나지 않습니다.

권한 사용 근거

본 확장 프로그램은 비디오 볼륨 제어 기능을 제공하기 위해 다음과 같은 권한을 요청합니다:

  • activeTab: 사용자가 현재 활성화된 탭에서만 확장 프로그램이 작동하도록 허용합니다. 이는 사용자가 확장 프로그램 아이콘을 클릭했을 때 현재 보고 있는 페이지의 비디오 요소에 접근하기 위함입니다.
  • scripting: 현재 활성화된 탭의 비디오 요소에 접근하여 볼륨을 조절하는 스크립트를 실행하기 위해 필요합니다.
  • 호스트 권한 (예: <all_urls>): 모든 웹사이트에서 비디오 볼륨을 제어할 수 있도록 허용합니다. 이는 사용자가 어떤 웹사이트에서든 비디오를 시청할 때 확장 프로그램이 작동할 수 있도록 하기 위함입니다. 이 권한은 데이터 수집 목적이 아닌, 기능 구현을 위한 최소한의 범위 내에서 사용됩니다.
